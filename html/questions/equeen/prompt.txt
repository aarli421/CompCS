Eight Queens Problem
====================

Eight Queens problem is the problem of placing 8 queens on an 8x8 chessboard
so that no queen is able to capture another queen by a regular chess queen move.
Print the number of possible solutions. One solution can be symmetric to or be a
mirror image of another solution.

PROBLEM NAME: equeen

INPUT FORMAT:

No Input

OUTPUT FORMAT:

* Line 1: Number of solutions in integer
